## Installation

```bash
npm install
```

## Set Up

```zsh
cp .env.example .env
```

Fill out all the env variables in `.env`.

```js
PRIVATE_KEY=
ALCHEMY_API_KEY=
MY_TKN_ADDRESS=
BALLOT_ADDRESS=
NETWORK=
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
